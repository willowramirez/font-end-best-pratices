### 常规 Git 分支

- dev
- master
- QA

### 临时 Git 分支

- 错误修复
- 热修复
- 功能分支
- 实验性分支
- WIP 分支

### 命名约定

1. 分支名称以组词开头
2. 在分支名称中使用唯一 ID
3. 使用连字符或斜杠作为分隔符
4. 带有作者姓名的 Git 分支 `<author>_<branch-type>_<branch-name>`
5. 避免只使用数字
6. 避免同时使用所有命名约定：一致性
7. 避免为长期存在的分支使用冗长的描述性名称
